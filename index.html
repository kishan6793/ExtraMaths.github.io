Advanced Encryption Standard (AES)

       Advanced Encryption Standard (AES) is a specification for the encryption of electronic data established by the U.S National Institute of Standards and Technology (NIST) in 2001.

       AES is widely used today as it is much stronger than DES and triple DES despite being harder to implement.

       AES is a block cipher.

 

 

1. Block cipher modes of operation:

 

       Encryption algorithms are divided into two categories based on the input type,

1.           block cipher

2.          stream cipher

       Block cipher is an encryption algorithm that takes a fixed size of input say b bits and produces a ciphertext of b bits again.

       If the input is larger than b bits, it can be divided further.

       Ex: Electronic code book (ECB), Cipher block chaining (CBC),Cipher feedback mode(CFB)  etc.

Introduction to AES

 

 

       AES uses 128 bits of plain text.

       AES uses 4 X 4 matrix as input each cell consists of 8 bits which means 1 byte each cell. In AES Algorithm word term is used widely. One word is equal to 4 byte (32 bits).

       AES consists of multiple rounds of processing different key bits like,

128 bits key- perform 10 rounds of encryption.

192 bits key- perform 12 rounds of encryption.

256 bits key- perform 14 rounds of encryption.

 

       With increase of 64 bits in Key, round increase by 2.

       Overall structures of AES encryption process shown in figure.

 


 

AES algorithm takes two inputs.

1. Plain text which is a 4 X 4 matrix.

2. Encryption key which is 4 X 4 matrix.

 

       For encryption, each round consists of the following four steps:

1.           Substitute Bytes-10 Rounds

2.          Shift Rows          -10 Rounds

3.          Mix Columns        -9 Rounds (not present in last round)

4.          Add Round Key   -10 Rounds

 

       The above steps are also called AES transformation functions.

 

       First, we have to expand our key to create add round key for each round.

1)       AES Sub Bytes (Substitute Bytes)

 

       First transformation function of AES is called as Substitute Bytes (Sub Bytes)

       It is also known as forward substitute bytes transformation.

       AES defines a 16x16 matrix of byte values, called an S-box, that contains a permutation of all possible 256 8-bit values.

       These row and column values serve as indexes into the S-box to select a unique 8-bit output value.

 

       The input matrix of sub bytes is XOR of plain text matrix and add round key matrix.

       The output matrix of Transformation function is called State array.

       State Array:

 

                                        

       How Substitute Bytes works?

  

                                                                                                             

       Each cell of State array consists of 8 bits.

       It is further divided into two groups of 4-bits each.

       The hexadecimal value of the first 4-bits will represent the row of S-box and the hexadecimal value of the rest 4-bits will represent the column of S-box.

       The value contained in the cell associated with that row and column will be the output cell value.

 

 

 

2)        AES Shift Rows

 

       The second transformation function of AES is called Shift Rows.

       It is also called forward shift raw transformation.

       State Array = Output of Substitute bytes.

       Rules of Shifting rows:

Row 1 -> No shifting

Row 2 -> 1-byte circular left shift

Row 3 -> 2-byte circular left shift

Row 4 -> 3-byte circular left shift

 

 

                      

 

 

 

3)       AES Mix Column

 

       The third transformation function of AES is called Mix Columns, operates on each column individually.

       State Array = Output of Shift Rows function

       Each byte of a column is mapped into a new value that is function of all four bytes in that column.

       For this step we need one predefined matrix.

       First, hexadecimal values are converted into binary numbers.

       This binary value is mapped with the Galois field (2^3).

       As per the Galois field the highest power of multiplication is 2^7 but we are getting 2^8.

       So, we have to use irreducible polynomial theorem,

X^8 = X^4 + X^3 + X + 1

4)       AES Add Round Key

 

       The fourth transformation function of AES is called Add Round Key.

       It is also known as forward add round key transformation.

       State Array = Output of Mix Column function

       EXOR function is performed for each byte of State Array and Round Key.
